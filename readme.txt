== Description ==

CC Recent Post is a simple recent post showing plugin in both sidebar or anywhere in your website.

Major Features:<br>
1. Simple Shortcode.<br>
2. Most Recent Post List.<br>
3. Most Commented Post List.<br>

How to use: <br>

For Showing Recent Post List: <br>

Shortcode: <pre>[ccrp]</pre> <br>
Options:
<ul>
	<li><b>per_page: </b><i>(default: 5)</i> To show number post.</li>
	<li><b>order: </b><i>(default: DESC)</i> Use to show the post order. <i>(values: DESC/ASC)</i></li>
	<li><b>img: </b><i>(default: true)</i> Use to show the post image thumbnail. <i>(values: true/false)</i></li>
	<li><b>img_type: </b><i>(default: sqr)</i> Use to show the post image thumbnail shape. <i>(values: sqr/circle)</i></li>
	<li><b>excerpt: </b><i>(default: false)</i> Use to show the post excerpt content. <i>(values: false/true)</i></li>
	<li><b>cat: </b><i>(default: true)</i> Use to show the post category name. <i>(values: true/false)</i></li>
</ul>
Full Shortcode : <pre>[ccrp pre_page="5" order="DESC" img="true" img_type="sqr" excerpt="false" cat="true"]</pre>
<br>

For Showing Most Commented Post List: <br>

Shortcode: <pre>[ccrp_mc]</pre> <br>
Options:
<ul>
	<li><b>per_page: </b><i>(default: 5)</i> To show number post.</li>
	<li><b>order: </b><i>(default: DESC)</i> Use to show the post order. <i>(values: DESC/ASC)</i></li>
	<li><b>img: </b><i>(default: true)</i> Use to show the post image thumbnail. <i>(values: true/false)</i></li>
	<li><b>img_type: </b><i>(default: sqr)</i> Use to show the post image thumbnail shape. <i>(values: sqr/circle)</i></li>
	<li><b>excerpt: </b><i>(default: false)</i> Use to show the post excerpt content. <i>(values: false/true)</i></li>
	<li><b>cat: </b><i>(default: true)</i> Use to show the post category name. <i>(values: true/false)</i></li>
</ul>
Full Shortcode : <pre>[ccrp_mc pre_page="5" order="DESC" img="true" img_type="sqr" excerpt="false" cat="true"]</pre>
<br>

Plugin Demo: http://www.codycave.com/Plugins/cc-recent-post/


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `CC Recent Post` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can use shortcode in page or post.
4. After applying the shortcode please go to the endless scroll plugin option (in your wordpress Dashboard Menu-bar) and click on 'Save Changes' button once.
